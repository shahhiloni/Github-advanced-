1. git --version : Shows installed Git version.
2. git init : Creates a new Git repository in your project folder.
3. git status : Shows files that are changed or not committed.
4. git add filename : Add a specific file.
5. git add . : Add all files. 
6. git commit -m "first commit" : Save changes with a message.
7. git remote add origin repository_link : Connect Local Repo to GitHub
8. git push origin main : Push Code to GitHub
9. git clone repository_link : Clone Repository
10. git pull origin main : Pull Latest Code
11. git push : add your command into repo
