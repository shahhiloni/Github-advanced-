1. Introduction to Git
- Git is a version control system used by developers to track changes in code.

2. Why Git is used
- Track code changes
- Work with multiple developers
- Go back to previous versions
- Manage project history

Example

If you are building a website or React project, Git records every change so you can restore old versions if something breaks.

3. Introduction to GitHub
- GitHub is an online platform that stores Git repositories.

4. What GitHub is used for

- Store projects online
- Collaboration with team members
- Backup code in the cloud
- Open-source project sharing

5. Difference Between Git and GitHub
Git - 
type : Version Control System
works : Locally on your computer
Purpose: Track code changes
Internet: Not required
Example : Commit, branch, merge

GitHub -
type : Cloud Hosting Platform
works : Online on the internet
Purpose: Store and share repositories
Internet: Required
Example : Pull request, issues, collaboration
