1. git branch branch_name : Creates a new branch.
2. git checkout branch_name  OR git switch branch_name : Switch Branch
3. git checkout -b branch_name : Create and Switch Branch Together
4. git merge branch_name : Merge another branch into current branch.
5. git log : Shows commit history in a simple format.
6. git diff : Shows what changes were made in files. (Show Differences Between Files)
7. git rm filename : Remove File from Git
8. git branch -m new_branch_name : Rename Branch
9. git branch -d branch_name : Delete Branch
10. git stash : Temporarily save changes without committing. (Stash Changes)
11. git revert commit_id : Undo a specific commit.
12. git reset --hard commit_id : Go back to previous commit.
13. git fetch : Downloads changes from GitHub but does not merge. (fetch latest Code)
14. git pull --rebase origin main : Pull with Rebase 
15. git remote -v : Show Remote Repository
